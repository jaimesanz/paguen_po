{% endblock %} {% block info_title %} Sobre las Transferencias {% endblock %} {% block info_content %} Los usuarios pueden realizar transferencias a sus compañeros. Esto resulta útil ya que, luego de ver el balance, los usuarios arreglan las cuentas por afuera del sistema, por lo que se hace necesario poder decirle al sistema que las cuentas fueron saldadas.

Al hacer una transferencia a otro usuario, se generan 2 Gastos: uno con saldo positivo (a nombre del usuario que transfirió) y otro negativo (a nombre del usuario al que le transfirieron). Estos Gastos deben ser confirmados por todos los usuarios antes de pasar a estar "pagados".

A diferencia de otros tipos de Gasto, los usuarios no tienen permitido editar transferencias.

Se reitera que para que el balance funcione correctamente, se deben realizar las transacciones pertinentes luego de arreglar las cuentas por afuera del sistema. {% endblock %}
